01 — Conceito filosófico // 01

De onde vem
essa estética?

A identidade não é decoração — é uma decisão filosófica sobre quem essa marca é no mundo. E ela carrega uma contradição proposital no DNA.

"Você não veio aqui para fazer a escolha. Você já a fez."
// Oráculo, Matrix Reloaded — citação de abertura do site

O site cita Matrix. A taróloga é programadora. A fonte de corpo é monospace. Não é coincidência — é postura. Essa marca existe na tensão entre o racional e o intuitivo, o código e o oráculo, o terminal e o cosmos. A estética reflete isso: dark, técnica, precisa — mas com calor humano nos detalhes. O cyan é a interface. O gold é o sagrado. O magenta é a urgência da ação.

// I
Terminal Místico
Parece uma interface de sistema. Mas o sistema lê cartas. O contraste é intencional — tecnologia como portal, não como produto.
// II
Luxo sem pedestal
Parece caro sem intimidar. O ouro é discreto, os cantos são retos, o tom é de conversa de igual pra igual.
// III
Escuro por postura
Não é dark mode por tendência. É porque tarot acontece no silêncio, no fundo da madrugada, quando a gente para de correr.
// IV
Glitch é honestidade
Scanlines, grain, glitch no logo — tudo diz: isso aqui não é polido além da conta. É real. É feito por uma pessoa.

// Quando usar cada cor de acento

>
Cyan — hover em links, bordas de componentes ativos, labels de seção, marcadores de lista, eyebrows, linhas decorativas, ícones de hover. É a cor do "sistema online".
>
Magenta — botão principal de CTA (sempre com clip-path angular), palavras em destaque dentro de headings (strong). Reservar para o momento de ação — não usar como decoração.
>
Gold — citações literárias (border-left), preços, a palavra em itálico que quebra o heading, linha superior de cards especiais. É o "sagrado" na interface técnica.
>
Muted — corpo de texto corrido, parágrafos de suporte, nav links em repouso. Passivo por natureza — não usar em nada interativo.
>
Ink — títulos, nomes de produto, destaques inline com <strong> dentro de parágrafos muted. O contraste ink-sobre-muted no mesmo parágrafo é assinatura visual do site.

04 — Elementos visuais // 04

O vocabulário
visual da marca.

Formas, texturas e motivos recorrentes que criam reconhecimento. Esses elementos juntos compõem a linguagem visual — use-os com consistência.

[ agendar ]
Clip-Path Angular
Botões CTA com canto cortado em 8px. Assinatura visual mais forte — representa a quebra entre o místico e o técnico. Inegociável em CTAs primários.
Scanlines
Linhas horizontais de 2px, 0.08 opacidade sobre toda a página. Textura de monitor CRT/terminal — essencial para a pegada cyberpunk.
Grid de Fundo
Grade de 48px × 48px em cyan com 3–5% de opacidade. Sempre presente — referência ao grid de sistemas e terminais de programação.
// 001
Linha Superior de Card
Todo card tem gradiente transparent → cyan → transparent no topo. É o "brilho" que diz: componente ativo. Em gold para cards especiais.
Anéis Rotativos
Três círculos concêntricos em velocidades e direções diferentes — cyan, gold, magenta. Representam o cosmos em movimento no sigil do hero.

Listas usam ">" em cyan. Referência ao prompt de terminal — o sistema está rodando, escutando, processando. Nunca usar bullet points.

08 — Regras visuais // 08

O que a marca faz.
E o que jamais faz.

Diretrizes que qualquer pessoa tocando no projeto precisa saber — designers externos, posts nas redes, qualquer asset visual.

✓  A marca faz isso
>
Fundo roxo-vinho escuro — nunca branco, cinza ou preto neutro
>
Duas fontes somente: Cormorant Garamond + Space Mono
>
CTAs com clip-path angular de 8px — inegociável
>
Prefixo "//" antes de números de seção e labels técnicas
>
Scanlines + grain em todos os fundos — sempre
>
Listas com marcador ">" em cyan — nunca bullet points
>
Citações com border-left: 2px solid --gold
>
Linha gradiente cyan no topo de cards ativos
>
Texto técnico nos cantos de containers grandes
>
Grid de fundo 48px × 48px em cyan com baixa opacidade
>
Cursor crosshair no desktop
✗  A marca jamais faz
✗
Fundo branco ou modo claro — não existe versão light
✗
Inter, Roboto, Montserrat, Poppins ou qualquer font genérica
✗
Border-radius generoso (8px+) — quebra o DNA angular
✗
Roxo/violeta como cor primária — cyan é o primário
✗
Emojis de estrela ⭐ lua 🌙 no UI — só em conteúdo
✗
Gradientes de arco-íris além do cyan e magenta
✗
Box-shadow duro e sólido sem blur
✗
Três cores de acento num único componente — max 2
✗
Texto uppercase em parágrafos corridos
✗
Fotos de banco de imagem com look wellness genérico
✗
Animações >600ms ou com bounce excessivo

// Movimento e animação

Entrada
fadeUp · 0.6–0.8s ease
Elementos revelam subindo 20px. Nunca caem, nunca saltam. É uma revelação — como virar uma carta.
Duração base
200ms–400ms · ease
Nada mais lento que 400ms (parece travado), nada mais rápido que 150ms (parece quebrado).
Stagger
animation-delay +0.1s
Grupos de elementos entram em cascata. Como cartas sendo viradas uma a uma.
Glitch
keyframes · loop 8s
Usado com parcimônia — só no logo do hero. Não replicar em múltiplos lugares.
Spin (anéis)
linear infinite · 8–20s
Lento o suficiente para ser hipnótico, não rápido para ser perturbador.
Scan line
top: -5% → 105% · 7s
A linha que percorre o cover. Opacidade 0.4 — sutil. Reforça que o sistema está ativo.
